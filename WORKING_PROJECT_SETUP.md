## 🔧 **TROUBLESHOOTING**

### **If you get errors:**
1. **Run database verification**: `npm run verify-db`
2. **Check environment variables**: Create `.env.local` file
3. **Clear node_modules**: `rm -rf node_modules && npm install`
4. **Use Windows fix**: `npm run fix-hydration-win`
